# Cognitive AI — Behavior & Thinking Pattern Analyzer

This project explores modeling human behavior patterns using AI.

Instead of analyzing static data, it focuses on how users interact, respond, and make decisions over time.

## Features

* Tracks user interaction patterns
* Analyzes response timing and behavior consistency
* Detects anomalies in decision patterns
* Builds a behavioral profile

## Goal

To understand and model human cognitive patterns using machine learning.

## Status

🚧 Experimental / in development

## Why this is interesting

Most AI systems analyze *what* users do.
This system focuses on *how* they think and respond.
